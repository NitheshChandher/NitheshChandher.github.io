Nithesh Chandher Karthikeyan

I am a Research Engineer, managing the technical side of the WASP research arena for Media and Language (WARA M&L). My main tasks are to design, implement and maintain machine learning platform on top of the OpenStack cloud infrastructure and assist with benchmarking machine learning challenges.

Recently, I completed my Master's degree in Computer Science at Uppsala University in Sweden. My Master thesis is about analysing visual political communication on YouTube videos using deep learning methods and is publicly available at this url. Check out my resume here.

WARA M&L Arena

At present, I am working on benchmarking the following challenges in Machine Leanring along with maintaining the WARA Media and Language platform.

VOT2022

Visual Object Tracking (VOT) challenge provides the platform to compare short term and long term visual trackers and discusses about the evaluation and advancements made in the field of visual tracking.

GENEA 2022

Generation and Evaluation of Non-verbal Behaviour for Embodied Agents (GENEA) challenge focuses on finding out the state of the art model in speech driven gesture synthesis.

ML Arena

The arena provides OpenStack services, along with GPU instances and cloud storage for the WASP community. For more information, check out the link.

Publications & Teaching

Analysis of visual political communication on YouTube

Master thesis project on developing deep learning methods using transfer learning for the analysis of image-based online communication. The methods include training a custom image classifier to detect object based labels related to climate change.

Intelligent Framework for Public Transport Bus Services system

A survey paper on smart city and its important features. The paper also discusses about the comparative analysis of how these features are implemented on major cities across the world. The article got published on Interational Journal of Recent Technology and Engineering (IJRTE).

Teaching Assistant for Data Mining Course (Sep 2020 - Mar 2021)

Supported IT and Engineering students for the Data Mining course during Autumn 2020 and Spring 2021. Provided assistance to students in executing different Data mining concepts using RapidMiner tool and monitor their progress in the project.


Projects

Decentralized air quality prediction using Federated Machine Learning

Developed a solution with Ericsson Research to predict the concentration of air pollutants (especially PM10) in the city of Stockholm using the Horizontal Federated Machine Learning approach. The code is publicly available here. Read more about the project in Ericsson blog.

Hand Gesture Recognition using Furhat Robot

A project on detecting different hand gestures using Deep Learning and create responsive behavior from Furhat Robot for each hand gesture. The code is available here.

Weather Station

The goal of this project is to build a weather station with real time notifications for climate monitoring, sync it with a cloud platform and analyse weather parameters using Raspberry Pi, Sense HAT, and Google Firebase. The project is available in github page.
